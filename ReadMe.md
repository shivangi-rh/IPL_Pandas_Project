# IPL Match Analysis

## Objective
The goal of this project is to analyze Indian Premier League (IPL) match dataset to understand
team performances, match results, and seasonal trends using Pandas.

## Dataset
Kaggle IPL Dataset


## Key Analysis
- Handled missing values
- Total matches per season
- Dominating team across season
- Analyzed trends in probablity of winning by chasing or fielding
- Grouped and aggregated data using `groupby`, `count`, `mode`, and sorting

##  Key Insights
- Most IPL matches are decided by wickets.
- Some matches result in 'no result' or 'abandonment'
- Some teams perform well across multiple seasons.
- Chasing has been trending decision over past few seasons.
